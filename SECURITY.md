# Security Policy

## Reporting a Vulnerability

We take the security of GigMarket seriously. If you find any security vulnerabilities or issues, please do not report them publicly via GitHub issues. Instead, please report them to the maintainers by sending an email to:

`chaaya.dev.1995@gmail.com`

Please include a detailed description of the issue, steps to reproduce it, and any potential remediation steps. We will acknowledge receipt of your report and work to resolve the issue as quickly as possible.
